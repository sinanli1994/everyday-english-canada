# Everyday English Canada

**Speak with confidence in everyday Canadian life.**

A small, friendly phrasebook website for newcomers, international students,
and visitors to Canada. It collects natural, polite English phrases — with
Chinese translations — for common everyday situations: grocery shopping,
eating out, visiting a doctor or pharmacy, and renting a home.

## Features

- 38 practical phrases across 4 categories: Grocery Store, Restaurant,
  Doctor & Pharmacy, and Renting a Home
- Category filter buttons plus a live search box that matches both
  English and Chinese text
- One-click **Copy English** button on every phrase card
- Helpful usage notes about Canadian customs (tipping, health cards,
  "first and last month's rent", and more)
- Fully responsive layout for desktop and mobile
- Accessible: semantic HTML, labelled controls, visible keyboard focus,
  and respect for `prefers-reduced-motion`

## Technologies

- Plain HTML, CSS, and JavaScript — no frameworks, no build tools,
  no dependencies

## Run it locally

Option 1 — just open the file:

1. Download or clone this folder.
2. Double-click `index.html`. It opens in your browser and everything works.

Option 2 — VS Code Live Server (auto-reloads on changes):

1. Open the folder in VS Code.
2. Install the "Live Server" extension.
3. Right-click `index.html` and choose **Open with Live Server**.

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub.
3. Click **Add New → Project** and import your repository.
4. Leave all settings as default (no framework, no build command) and
   click **Deploy**.
5. Vercel gives you a public URL you can share right away.
